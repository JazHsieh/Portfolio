Artifact 01

Machine Learning Algorithm Visual Framework

A comprehensive visual reference mapping 10 machine learning algorithms across 4 major AI/ML domains, organized by learning paradigm, application area, real-world use cases, and core mechanics.

Domain

AI / ML

Type

Visual Infographic

Format

HTML

Objective

To develop a structured visual understanding of how machine learning algorithms are categorized by learning style and application domain, enabling informed algorithm selection for real-world tasks.

Description

The framework covers 10 algorithms spanning tabular data (Decision Tree, Random Forest, Gradient Boosting, SVM, K-Means), computer vision (CNN, PCA), NLP (Transformer/BERT, Linear Regression), and generative AI (GAN, Diffusion Model). Each entry includes its type, domain, use cases, and explanation.

Process

I began by reviewing course materials on supervised and unsupervised learning, then identified representative algorithms for each of the four application domains covered in the module. For each algorithm, I researched its core mechanism and mapped it to concrete real-world use cases to ensure the framework would be useful beyond the classroom. The visual layout was designed to communicate both the learning paradigm (via color-coded badges) and the domain structure (via grouped sections), making comparisons intuitive at a glance.

Tools and Technologies Used

HTML5 / CSS3 Google Fonts Visual Design Principles AI/ML Research Information Architecture

References

Course module: Tabular Data, Computer Vision, NLP, and Generative AI

Unique Value

This artifact consolidates 10 algorithms into a single visual reference that would take hours to assemble from scattered textbooks. It demonstrates my ability to synthesize technical content into a clear, accessible format that professionals and learners can use immediately.

Relevance

Algorithm selection is one of the most critical early decisions in any ML project. By grounding each algorithm in its ideal domain and real use cases, this framework is directly applicable to practitioners evaluating tools for business problems in industry settings.

Tabular Data 5 algorithms
Decision TreeSupervised

How it works

Recursively splits data using feature thresholds to form if-then rules at each leaf node.

Use Cases

Credit approval, medical diagnosis, customer churn prediction.

Random ForestSupervised

How it works

Builds hundreds of decision trees on random subsets, aggregating their votes to reduce overfitting.

Use Cases

Fraud detection, stock prediction, disease classification.

Gradient BoostingSupervised

How it works

Sequentially trains weak learners, each correcting the previous one's errors by optimizing a loss function.

Use Cases

Sales forecasting, Kaggle competitions, click-through rate prediction.

Support Vector MachineSupervised

How it works

Finds the optimal hyperplane maximizing margin between classes; uses kernels for non-linear data.

Use Cases

Spam filtering, bioinformatics, handwriting recognition.

K-Means ClusteringUnsupervised

How it works

Assigns data to K clusters by minimizing distance between each point and its centroid until convergence.

Use Cases

Customer segmentation, document grouping, anomaly detection.

👁
Computer Vision 2 algorithms
Convolutional Neural NetworkSupervised

How it works

Applies learnable filters across image regions to extract hierarchical features from edges to high-level objects.

Use Cases

Image classification (ResNet), object detection (YOLO), medical imaging, autonomous vehicles.

PCAUnsupervised

How it works

Computes orthogonal components capturing maximum variance, compressing high-dimensional data.

Use Cases

Image compression, face recognition preprocessing, noise reduction, visualization.

💬
Natural Language Processing 2 algorithms
Transformer (BERT)Supervised

How it works

Self-attention weighs every word against every other word, enabling rich bidirectional context understanding.

Use Cases

Sentiment analysis, question answering, named entity recognition, search ranking.

Linear RegressionSupervised

How it works

Models relationships between features and a continuous output by fitting a hyperplane minimizing squared residuals.

Use Cases

Price prediction, trend analysis, feature importance scoring in NLP pipelines.

Generative AI 2 algorithms
Generative Adversarial NetworkUnsupervised

How it works

A Generator and Discriminator compete in a minimax game until the generator produces indistinguishable outputs.

Use Cases

Synthetic image generation, deepfake detection, data augmentation, style transfer.

Diffusion ModelUnsupervised

How it works

Adds noise progressively during training, then learns to reverse that process to generate new data from pure noise.

Use Cases

Stable Diffusion, DALL-E image synthesis, drug molecule design, audio generation.
